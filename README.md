Final project for Dr. Dechevs Parallel and Concurrent Programming class. The team is: Arjun Pherwani, Cameron Cuff, Dylan Skelly, Joseph Giordano.

To run type:
```

```
from the root of the repository.

```
cargo test
```
from the root of the repository.
